17

Clarification about the different languages :
35
English: American English, only available if the region is set to The Americas (internally USen).
36
English (Europe): English version available for all other regions (internally EUen).
37
German: German version of the game, available worldwide (internally EUde).
38
Spanish: Spanish version available for all other regions except The Americas (internally EUes).
39
Spanish: American Spanish, only available if the region is set to The Americas (internally USes).
40
French: French version available for all other regions except The Americas (internally EUfr).
41
French: Canadian French, only available if the region is set to The Americas (internally USfr).
42
Italian: Italian version of the game, available worldwide (internally EUit).
43
Dutch: Dutch version of the game, available worldwide (internally EUnl).
44
Chinese: Simplified Chinese (mainland China/Singapore) version of the game, available worldwide (internally CNzh).
45
Chinese (Traditional): Traditional Chinese (Hong Kong/Taiwan) version of the game, available worldwide (internally TWzh).
46
Japanese: Japanese version of the game, available worldwide (internally JPja).
47
Korean: Korean version of the game, available worldwide (internally KRko).
48
Russian: Russian version of the game, available worldwide (internally EUru).
